ℹ️ Acerca del Sistema

Este dashboard muestra los registros OCR y ahora también las sesiones de precaptura para revisar cédulas antes de capturarlas definitivamente a la base. Los datos se almacenan en PostgreSQL y se actualizan desde la web y Telegram.

🤖 Características:

  • OCR local y revisión previa a captura
  • Validación automática de nombres mexicanos
  • Verificación humana para casos de baja confianza
  • Almacenamiento persistente en PostgreSQL
  • Dashboard web en tiempo real
